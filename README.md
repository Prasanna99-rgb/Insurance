* 📥 Download Report (PDF)
* 🎨 Beautiful UI
* 📱 Responsive Layout

## 🧠 Machine Learning Model

* Algorithm: Linear Regression
* Features Used:

  * Age
  * BMI
  * Number of Children
  * Smoker Status
