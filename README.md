## 📌 Overview

Insurance Cost Prediction App is a Machine Learning-powered web application developed using **Streamlit**. The application predicts medical insurance costs based on user-provided information such as age, BMI, smoking status, and number of children.

This project demonstrates the complete Machine Learning workflow, including data preprocessing, model training, deployment, and user interaction through a modern web interface.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Prasanna99-rgb/insurance-app.git
cd insurance-app
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
